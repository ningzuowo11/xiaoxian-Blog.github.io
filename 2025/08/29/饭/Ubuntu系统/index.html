系统安装
准备一个iso系统文件，一个30G左右的U盘，拷贝软件balenaEtcher(可以在ubuntu官网找到)，以及解除写保护的神器Diskgenius。
将下载好的iso文件通过balena拷贝进入U盘，因为拷贝的时候会格式化U盘建议先备份重要文件，如果遇到写保护，可以用diskgenius删除U盘下的的所有分区，然后在新建并保存一个分区。
装机必备--搜狗输入法
1、添加中文语言支持
首先打开系统设置——区域和语言——管理已安装的语言——在“语言”tab下——点击“添加或删除语言”（正常情况下初次登录系统会引导你安装），接着回到“语言支持”窗口，在键盘输入法系统中，选择“fcitx”，并应用到整个系统。
2、通过命令行安装搜狗输入法
sudo dpkg -i sogoupinyin_版本号_amd64.deb
sudo apt -f install
3，注销计算机即可正常使用搜狗输入法
a，更新源，在终端执行
sudo apt update
b，安装fcitx输入法框架，因为搜狗是基于fcitx开发的，如果不装就不能正常运行
sudo apt install fcitx
c，设置fcitx为系统输入法
d，设置fcitx开机自启动
sudo cp /usr/share/applications/fcitx.desktop /etc/xdg/autostart/
e，卸载系统ibus输入法框架
sudo apt purge ibus
4、安装搜狗输入法
安装输入法
sudo dpkg -i 安装包
安装依赖
sudo apt install libqt5qml5 libqt5quick5 libqt5quickwidgets5 qml-module-qtquick2
sudo apt install libgsettings-qt1
解决搜狗老是闪烁无法正常使用
将#Wayland前面的#删去即可
sudo nano /etc/gdm3/custom.conf如果安装了vim也用vi。
安装docker
sudo apt install docker.io
几个常用的指令 
<
sudo systemctl status docker 查看运行状况

docker pull 拉取镜像

docker images 列出镜像

docker run 运行容器

docker stop 列出容器

docker start 停止容器

docker exec -it 启动容器

docker logs -f 进入容器

docker rm 删除容器

docker rmi 删除镜像
RustDesk远程控制（熄屏状态下也可以）
彻底摆脱会员制
官网https://github.com/rustdesk/rustdesk/releases，我用的是UbuntuX86版本
接着
sudo apt update
sudo apt install ./rustdesk-1.2.3-2-x86_64.deb
Ubuntu默认的显示器程序是wayland，在息屏状态下远程软件无法唤醒屏幕。为了能唤醒屏幕我们需要改成xorg。只需要把#Waylang前的#删了
sudo vi /etc/gdm3/custom.conf
接着，修改rustdesk设置设置->安全->允许IP直接访问，使用默认端口，这样在家里就可以不使用中继服务器，而是IP直连，提高速度并去设置固定密码。
最后在搞网络设置ID和中继填自己的服务器ip，端口在21116到21119之间
安装SSH-server
sudo apt openssh-server
控制方，在终端输入ssh 被控用户名@被控用户ip
